Part Analyzer Alpha

Step 1: Download the zip file of the application. The included folder includes all the binaries, and does not install as a proper application. Instead, you simply run the application by double clicking the shortcut "MedemaPartAnalyzer - Shortcut". Note that the ease of not installing the software makes Windows nervous and it'll warn you the first time you run it. Rest assured, the program only opens files you give it and saves files.

Step 2: Watch the following instructional video.

Step 3: Run the software. We have included five example parts that can be used in the main folder, but feel free to try additional ones.

Step 4: Email us your impressions at beta@medemalabs.com, fill in our survey, or chat on our gitter page. If you run into errors, please send us back the .log files that are created in the bin directory
